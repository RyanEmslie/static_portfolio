SpaceX-Apollo-GraphQL

This project connects to SpaceX open API. GraphQL is used to only call the specific queries from the SpaceX API. React is used on the frontend to render the different components of the website. This was my first attempt at using Apollo to manage local state. While Apollo may not be the best use case for a project this size, I want to get a feel for the syntax and usability.
