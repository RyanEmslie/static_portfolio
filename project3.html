Dynamic Library Manager

This project creates a library managment system web application that dynamically displays database entries. The web application allows functionality for listing, adding, updating, and deleting books, using JavaScript, Node.js, Express, Pug, and the SQL ORM Sequelize. If I were to rewrite this application I would use React library in place of the Pug templating engine. The final app is deployed on Heroku.
